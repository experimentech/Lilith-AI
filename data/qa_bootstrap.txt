# ============================================================================
# MACHINE LEARNING
# ============================================================================

Q: What is machine learning?
A: Machine learning is a subset of AI where computers learn patterns from data without being explicitly programmed for every scenario.

Q: How does machine learning work?
A: Machine learning works by training algorithms on data, allowing them to identify patterns and make predictions or decisions based on new input.

Q: What is supervised learning?
A: Supervised learning is when algorithms learn from labeled training data, where the correct answers are provided for the model to learn from.

Q: What is unsupervised learning?
A: Unsupervised learning finds patterns in unlabeled data without predefined categories, discovering hidden structures on its own.

Q: What is reinforcement learning?
A: Reinforcement learning trains agents to make decisions by rewarding desired behaviors and penalizing undesired ones, learning through trial and error.

Q: What are machine learning applications?
A: Machine learning powers recommendation systems, image recognition, natural language processing, autonomous vehicles, and many other applications.

Q: What is training data?
A: Training data is the dataset used to teach a machine learning model, containing examples from which the algorithm learns patterns.

Q: What is a machine learning model?
A: A machine learning model is the mathematical representation learned from training data that can make predictions on new data.

Q: What is overfitting?
A: Overfitting occurs when a model learns the training data too well, including noise, reducing its ability to generalize to new data.

Q: What is underfitting?
A: Underfitting happens when a model is too simple to capture the underlying patterns in the data, resulting in poor performance.

# ============================================================================
# NEURAL NETWORKS
# ============================================================================

Q: What are neural networks?
A: Neural networks are computing systems inspired by biological neural networks, consisting of interconnected nodes that process information.

Q: How do neural networks work?
A: Neural networks work by passing input through layers of interconnected nodes, each applying mathematical transformations to produce an output.

Q: What is a neuron in neural networks?
A: A neuron is a computational unit that receives inputs, applies weights and an activation function, and produces an output signal.

Q: What are layers in neural networks?
A: Layers are groups of neurons in a neural network: input layers receive data, hidden layers process it, and output layers produce results.

Q: What is deep learning?
A: Deep learning uses neural networks with many layers (deep networks) to learn complex patterns from large amounts of data.

Q: What is backpropagation?
A: Backpropagation is the algorithm used to train neural networks by calculating gradients and adjusting weights to minimize error.

Q: What is an activation function?
A: An activation function determines whether a neuron should activate, introducing non-linearity to help networks learn complex patterns.

Q: What are weights in neural networks?
A: Weights are parameters that determine the strength of connections between neurons, adjusted during training to improve performance.

Q: What is a convolutional neural network?
A: A convolutional neural network (CNN) is designed for processing grid-like data such as images, using convolutional layers to detect features.

Q: What is a recurrent neural network?
A: A recurrent neural network (RNN) processes sequential data by maintaining hidden states, making it useful for time series and natural language.
