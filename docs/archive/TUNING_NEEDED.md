# Tuning Required

This document lists areas that require parameter tuning or additional validation. Examples and conversational snippets have been removed and replaced with actionable items.

- Similarity thresholds for embedding-based retrieval.
- Hybrid weighting between keyword and semantic matching.
- Reinforcement parameters for response selection rewards.

Action items:

- Log scoring breakdowns to aid threshold selection.
